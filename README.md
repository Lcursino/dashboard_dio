# README: Dashboard de Vendas de Assinaturas Xbox Game Pass

![image](https://github.com/user-attachments/assets/ea707c6f-a077-4b70-b3b9-a8a72a304c1b)


## Visão Geral
Este documento fornece uma explicação dos dados e do dashboard para as vendas de assinaturas do Xbox Game Pass. O dashboard foi projetado para fornecer uma visão clara da receita das assinaturas, status de renovação e vendas específicas de passes.

## Descrição dos Dados

### Fonte dos Dados
Os dados usados neste dashboard vêm de um arquivo chamado `Pasted_Text_1742338973298.txt`. Ele contém informações detalhadas sobre os assinantes, incluindo:

- **ID do Assinante**: Identificador único para cada assinante.
- **Nome**: Nome do assinante.
- **Plano**: Tipo de plano de assinatura (Ultimate, Core, Standard).
- **Data de Início**: Data em que a assinatura começou.
- **Renovação Automática**: Indica se a assinatura está configurada para renovação automática.
- **Preço da Assinatura**: Custo mensal da assinatura.
- **Tipo de Assinatura**: Frequência de pagamento (Mensal, Trimestral, Anual).
- **EA Play Season Pass**: Se o assinante possui um EA Play Season Pass.
- **Preço do EA Play Season Pass**: Preço do EA Play Season Pass.
- **Minecraft Season Pass**: Se o assinante possui um Minecraft Season Pass.
- **Preço do Minecraft Season Pass**: Preço do Minecraft Season Pass.
- **Valor do Cupom**: Valor de qualquer cupom aplicado.
- **Valor Total**: Valor total da assinatura, incluindo passes e cupons.

## Explicação do Dashboard

### Período de Cálculo e Data de Atualização
- **Período de cálculo:** 01/01/2024 - 31/12/2024
- **Data de atualização:** 25/12/2024 09:00:00

### Métricas Principais

#### Receita Total EA Play Season Pass
- **Valor:** R$ 990,00
- Esta métrica mostra a receita total gerada por assinantes que possuem um EA Play Season Pass.

#### Receita Total Minecraft Season Pass
- **Valor:** R$ 1.140,00
- Esta métrica mostra a receita total gerada por assinantes que possuem um Minecraft Season Pass.

#### Receita Total Xbox Game Pass
- **Sim:** R$ 1.502,00
- **Não:** R$ 806,00
- Este gráfico de barras compara a receita total de assinantes com renovação automática habilitada versus aqueles sem. A categoria "Sim" representa assinantes cujas assinaturas estão configuradas para renovação automática, enquanto a categoria "Não" representa aqueles que não têm renovação automática habilitada.

### Contagem de Renovações Automáticas
- O gráfico de pizza à direita no dashboard mostra a distribuição das assinaturas com base no status de renovação automática. A seção verde representa a porcentagem de assinaturas configuradas para renovação automática, enquanto a seção cinza representa aquelas que não estão.

## Navegação e Filtros
Na barra lateral esquerda, você pode filtrar os dados por **Tipo de Assinatura**:
- **Anual**
- **Mensal**
- **Trimestral**

Selecionar uma dessas opções atualizará as métricas do dashboard de acordo, permitindo analisar os dados com base em diferentes frequências de assinatura.

## Conclusão
Este dashboard oferece uma visão abrangente das vendas de assinaturas do Xbox Game Pass, focando em métricas-chave como receita dos passes EA Play e Minecraft, receita total de assinaturas com e sem renovação automática e a distribuição geral do status de renovação automática. Usando os filtros disponíveis, os usuários podem obter insights mais profundos sobre o desempenho de diferentes tipos de assinatura.
